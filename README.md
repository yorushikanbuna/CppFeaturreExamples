# C++ Feature Examples

This project demonstrates key features introduced in C++11, C++14, C++17, and C++20 standards. Each directory contains self-contained examples with explanations, perfect for interview preparation.

## Project Structure

- `cpp11/` - Examples of C++11 features
- `cpp14/` - Examples of C++14 features
- `cpp17/` - Examples of C++17 features
- `cpp20/` - Examples of C++20 features

## How to Build

```bash
mkdir build
cd build
cmake ..
make
```

Each example compiles to its own executable. Run any example with `./<example-name>`.

## Purpose

This project is designed to help developers:
- Understand modern C++ features through practical examples
- Prepare for technical interviews requiring C++ knowledge
- See real-world usage of language features
- Compare implementations across different C++ standards
